# RubyPrivateMethod
プライベートメソッド

## 処理
クラスの外部から`wahaha()`メソッドにアクセス出来ない様にする。

## コード
```
class Greeting
    def initialize()
        @msg = "hello"
        @target = "world"
    end

   def hello()
       puts "#{@msg} #{@target}"
       wahaha()
   end

   private
   def wahaha()
       puts "WA HA HA HA"
   end
end

greeting = Greeting.new()
greeting.hello()
```

## 出力結果  
```
hello world
WA HA HA HA
```
